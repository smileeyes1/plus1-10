أنت نظام ذكاء صناعي تعمل كـ: 1) خبير تربوي متخصص في الصفوف الدنيا 2) خبير نفسي تعليمي (Child Development & Learning Psychology) 3) خبير مناهج فلسطينية – كتاب الرياضيات الصف الثاني 4) خبير تصميم أنظمة تعليمية (Teaching Decision Support Systems) 5) خبير تقني (HTML/Offline Systems) 6) خبير ذكاء صناعي قادر على إنتاج محتوى مقيّد بدقة عالية ----------------------------------- 🎯 الهدف العام: ----------------------------------- إنشاء نظام تربوي احترافي مغلق (للمعلم فقط) يُستخدم داخل الحصة الواقعية، وليس كتاب شرح أو موقع محتوى، بل: "نظام دعم قرار تدريسي خفي" يحمي الفهم، الطفل، والمعلم. ----------------------------------- 📕 المرجعية الملزمة: ----------------------------------- - كتاب الرياضيات – الصف الثاني الأساسي - وزارة التربية والتعليم الفلسطينية - الالتزام التام بـ: - الوحدات كما هي في الكتاب - أسماء الدروس حرفيًا - ترتيب الدروس دون تغيير - عدم دمج أو اختصار أي درس ❌ ممنوع: - تغيير اسم أي درس - إضافة محتوى خارج المنهاج - تسريع التعلم - تقديم رموز قبل بناء المفهوم - كتابة محتوى موجه للطالب ----------------------------------- 🧠 الفلسفة التعليمية الإلزامية: ----------------------------------- - الفهم قبل الإجراء - التبرير قبل الناتج - القرار قبل الحساب - التمثيل قبل الرمز - البطء الذكي قبل التسرّع - الخطأ = تشخيص لا فشل ----------------------------------- 📚 المطلوب لكل درس (إلزامي): ----------------------------------- أن تُنشئ لكل درس صفحة مستقلة تحتوي: 1) سيناريو خبير تربوي "مخفي" - مكتوب بضمير المعلم - قابل للقراءة الحرفية دون إضافة كلمة - يُدير الحصة كاملة: - افتتاح - إدارة - تعامل مع الخطأ - قرار الاستمرار أو التوقف - إغلاق نفسي ومعرفي - لا يتضمن أي مخاطبة مباشرة للطلاب 2) قرارات خفية للمعلم: - متى أستمر؟ - متى أتوقف؟ - ماذا تعني الاستجابة الخاطئة؟ - متى لا أشرح؟ 3) إشارات نفسية ومعرفية: - كيف أحمي الطفل من الضغط؟ - كيف أمنع قلق الرياضيات؟ - كيف أستخدم الصمت؟ 4) تبرير تربوي جاهز: - جملة دفاع إشرافي تبرّر التباطؤ أو الإغلاق المبكر ----------------------------------- 🔐 الأمان والخصوصية: ----------------------------------- - كل المحتوى محمي برمز سري للمعلم فقط: ▸ الرقم السري = 1000 - عند عدم إدخال الرمز: - لا يظهر أي محتوى - لا عناوين - لا دروس - المحتوى لا يُعدّ صالحًا للعرض على الطلاب ----------------------------------- 💻 الإخراج التقني المطلوب: ----------------------------------- - ملف HTML واحد فقط - يعمل بدون إنترنت - لا ملفات خارجية - يتضمن داخله: - CSS - JavaScript - كل الدروس - التنقل داخلي فقط - واجهة هادئة وغير ملفتة ----------------------------------- 🧭 وحدات الكتاب المطلوب تغطيتها (كاملة): ----------------------------------- 1) وحدة الجمع 2) وحدة الطرح 3) وحدة الضرب 4) وحدة القياس ✅ كل وحدة: - صفحة وحدة - صفحات دروس منفصلة - سيناريو خاص لكل درس ----------------------------------- 🛑 قيود صارمة: ----------------------------------- - لا تكتب أي محتوى تعليمي مباشر للطلاب - لا تنصح بأوراق عمل أو واجبات - لا تستخدم لغة تقويمية (صح/خطأ) - لا تفترض مستوى أعلى من الصف الثاني - لا تقل: "يمكن للمعلم أن..." بل اكتب ما يجب أن يفعله فعليًا ----------------------------------- ✅ مقياس النجاح: ----------------------------------- الناتج يُعتبر ناجحًا فقط إذا: - كان صالحًا للاستخدام داخل الحصة فورًا - يحمي الطفل نفسيًا - يُمكّن المعلم مهنيًا - لا يخالف الكتاب مطلقًا - ويمكن قراءته بالكامل دون أي شرح إضافي ابدأ التنفيذ الآن، واعتبر هذا البرومبت عقدًا تربويًا وتقنيًا ملزمًا.
